¿Qué es jQuery?

Es una librería para Javascript concisa y eficiente, que tiene como principales objetivos simplificar:

  • El acceso a los elementos del documento HTML.
  • El manejo de eventos,
  • Las animaciones y los efectos,
  • Las interacciones AJAX.

Características Principales

  • Selectores
      Elimina la necesidad de getElementById()
  • Operaciones sobre conjuntos de elementos.
  • Eventos.
  • Extensible a través de Plugins.

Selectores

$(".tabla") // por clase de CSS
$("#titulo") // por ID
$("ul") // por nombre de tag
$("ul li") // elemento descendiente
$("ul").find("li") // más elementos desc.
$("li", lista) // dentro de otros elementos
$("ul > li") //elemento hijo (desc. directo)
$("input:checkbox") // filtrar
$("a[href=#Home]") // atributos
$("a[href$=.jpg]") // atributos que terminan en

Conjuntos de Elementos

  • Cada selector devuelve un conjunto de elementos (un objeto que actúa como un arreglo).
  • La sentencia: 
    • $(".tabla td").css("background","red");
  • Modifica el fondo de todos los elementos devueltos por el selector.

Conjuntos de Elementos

  • Posibles operaciones:
    • .html() .text() .val() .attr() .append() .empty()
    • .css() .addClass() .removeClass() .hasClass() .offset() .height() .width() .scrollTop() .scrollLeft() .show() .hide()
    • .find() .is() .has() .not() .filter() .parent() .closest() .next()
    • .live() .bind() .click() .dblclick() .hover() .toggle() .blur() .keydown() .keyup() .resize() .mouseover() .mousedown()
  • Encadenando Operaciones
    • $("th").css("cursor","pointer").click(headerClicked);

Eventos

  • Es posible asociar eventos facilmente desde el script.
  • Evita agregar los atributos de eventos en el documento HTML, mejorando la reutilización y la claridad del código.
    • $(document).ready(function() {  
  $("#myLink").click(function(e) { 
    e.preventDefault();
    $(".myDiv").show();
  });
});

Eventos y Callbacks

    $("#myLink").click(function(e) { 
  $(".myDiv").show();
});
    $("#myLink").click(myLinkClick);

function myLinkClick(e) { 
  $(".myDiv").show();
});
    $("#myLink").toggle(function(e) { 
  $(this).addClass("selected");
}, function(e) { 
  $(this).removeClass("selected");
});

Utilizando jQuery

  • Descargar la última versión de jQuery (actualmente 2.2.2).
  • Incluirla en el documento HTML:
    • <script language="javascript" src="jquery-2.2.2.min.js"></script>
  • Si se desea utilizar jQuery UI, hay que descargar los archivos correspondientes e incluirlos de manera similar.
  • Otra posibilidad es utilizar directamente los servidores de jQuery para incluirlo:
    • <script language="javascript" 
      src="https://code.jquery.com/jquery-2.2.2.min.js"></script>
