¿Qué es jQuery?

Es una librería para Javascript concisa y eficiente, que simplifica muchas tareas comunes de JS que normalmente requieren muchas líneas de código.

jQuery permite y/o facilita:

  • manipulación HTML/DOM,
  • manipulación CSS,
  • manejo de eventos HTML,
  • animaciones y efectos,
  • interacciones AJAX.

Sintaxis

La sintaxis jQuery está hecha a medida para seleccionar elementos HTML y realizar alguna acción en el/los elemento/s.

    $(selector).acción()
  • El signo $ para definir acceder a jQuery,
  • un (selector) para seleccionar elementos HTML, y
  • una acción() de jQuery para realizar sobre el/los elemento/s.

Selectores

    $("ul") // por nombre de elemento
$(".class") // por clase de CSS
$("#id") // por ID
    $(*) // todos los elementos
$(this) // el elemento actual
$("ul li") // elemento descendiente
$("ul").find("li") // más elementos desc.
$("li", lista) // dentro de otros elementos
$("ul > li") //elemento hijo (desc. directo)
$("input:checkbox") // filtrar
$("a[href=#Home]") // atributos
$("a[href$=.jpg]") // atributos que terminan en

Conjuntos de Elementos

  • Cada selector devuelve un conjunto de elementos (un objeto que actúa como un arreglo).
  • Por ejemplo: 
    • $(".tabla td").css("background","red");
  • Modifica el fondo de todos los elementos devueltos por el selector.
  • También se pueden encadenar operaciones:
    • $("th").css("cursor","pointer").click(headerClicked);

Eventos

Todos las posibles acciones a las cuales una página web puede responder son denominados eventos. Con jQuery:

  • Es posible asociar eventos facilmente desde el código.
  • Evita agregar los atributos de eventos en el documento HTML, mejorando la reutilización y la claridad del código.
    • $(document).ready(function() {  
  $("#myLink").click(function(e) { 
    e.preventDefault();
    $(".myDiv").show();
  });
});
  • Eventos mas comunes: 
    • .click() .dblclick() .hover() .focus() .blur() .mouseenter()

Eventos y Callbacks

    $("#myLink").click(function(e) { 
  $(".myDiv").show();
});
    $("#myLink").click(myLinkClick);

function myLinkClick(e) { 
  $(".myDiv").show();
});
    $("#myLink").toggle(function(e) { 
  $(this).addClass("selected");
}, function(e) { 
  $(this).removeClass("selected");
});

Manipulación del DOM

  • Posibles operaciones:
    • Obtener y modificar:
      • .html() .text() .val() .attr()
    • Agregar
      • .append() .prepend() .after() .before()
    • Eliminar
      • .remove() .empty()
    • CSS
      • .addClass() .removeClass() .toggleClass() .hasClass() .css()

Manipulación del DOM

  • Dimensiones 
    • .height() .width() .innerWidth() .innerHeight() .outerWidth() .outerHeight()

Navegando el DOM

  • Posibles operaciones:
    • Ancestros
      • .parent() .parents() .parentsUntil()
    • Descendientes
      • .children() .find()
    • Primos
      • .siblings() .next() .nextAll() .nextUntil() .prev() .prevAll() .prevUntil()
    • Filtros
      • .first() .last() .eq() .filter() .not()

Ajax - Load

Sintaxis

$(selector).load(URL, data, callback);

Ejemplo

$("button").click(function(){
    $("#div1").load("demo.txt", function(responseTxt, statusTxt, xhr) {
        if (statusTxt == "success")
            alert("contenido cargado exitosamente!");
        if (statusTxt == "error")
            alert("Error: " + xhr.status + ": " + xhr.statusText);
    });
});

Ajax - Get and Post

Sintaxis

$.get(URL, callback);
$.post(URL, data, callback);

Ejemplo

$("button").click(function() {
    $.post("/sendData",
    {
        materia: "IAW",
        universidad: "UNS"
    },
    function(data, status){
        alert("Data: " + data + "\nEstado: " + status);
    });
});

Utilizando jQuery

  • Descargar la última versión de jQuery (actualmente 3.3.1).
  • Incluirla en el documento HTML:
    • <script language="javascript" src="jquery-3.3.1.min.js"></script>
  • Otra posibilidad es utilizar directamente los servidores CDN de jQuery o Google para incluirlo:
    • <script language="javascript" 
      src="https://code.jquery.com/jquery-3.3.1.min.js"></script>
    <script language="javascript" 
      src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>
