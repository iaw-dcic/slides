Heroku: Instalación

Instalar Heroku

Asegurarse que funcione: 

heroku

Loguearse: 

heroku auth:login

Actualizar el Proyecto

Actualizar package.json agregando: 

"engines": {
    "node": "^9.4.0",
    "npm": "^5.6.0"
  },

Agregar el archivo procfile con el contenido: 

web: npm start

Subirlo a Heroku

Crear aplicación en Heroku: 

heroku create

Renonbrar la aplicación en Heroku: 

heroku apps:rename salspizza

Subir el código a Heroku: 

git push heroku master
git push heroku branch:master

MLab: Hosting para Mongo

Conectarse por terminal: 

mongo ds113700.mlab.com:13700/<db> -u <dbuser> -p <dbpassword>

Configurar la variable de entorno: 

heroku config:set MLAB_URI=mongodb://<dbuser>:<dbpassword>@ds113700.mlab.com:13700/<db>

Actualizar el código de conexión: 

const dbURI = process.env.MLAB_URI;

MLab: Hosting para Mongo

Dump de la base de datos local 

mongodump -h localhost:27017 -d <db>

Restore en la base de datos en MLab 

mongorestore -h ds113700.mlab.com:13700 -d <db> -u <user> -p <pass> dump/
