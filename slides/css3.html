Box Sizing

Por defecto, el ancho y alto se calculan asi:

  • ancho + padding + borde = ancho real
  • alto + padding + borde = alto real

box-sizing permite incluir en el ancho y alto el padding y el borde.

Usando:

* {
    box-sizing: border-box;
}

el padding y el borde es considerado parte del ancho y alto.

Flex Box

  • Flexbox es un nuevo modo de layout introducido en CSS3.
  • Usar flexbox permite asegurar que los elementos se comporten de manera predecible cuando la disposición de la página se debe acomodar a los distintos tamaños de pantallas y tipos de dispositivos.
  • Consiste en Contenedores y Elementos
  • Dentro de un Contenedor hay uno o más Elementos
  • Las propiedades flex no afectan a los elementos fuera del Contenedor ni dentro de los Elementos

Flex Box

.flex-container {
    display: flex;
    flex-wrap: wrap;
    align-content: center;
    width: 300px;
    height: 300px;
    background-color: lightgrey;
}
.flex-item {
    background-color: cornflowerblue;
    width: 120px;
    height: 100px;
    margin:5px auto;
    font-size: 20px;
}

Gradientes

CSS3 permite permiten mostrar transiciones suaves entre los dos o más colores

Previo a CSS3, para lograr el mismo efecto, era necesario utilizar imágenes, lo cual afectaba el rendimiento de la página.

CSS3 define dos tipos de gradientes:

  • Los gradientes lineales (hacia arriba / abajo / izquierda / derecha / diagonal)
  • Los gradientes radiales (definido por su centro)
  #grad1 {
    background: linear-gradient(141deg, #0fb8ad 0%, 
                                        #1fc8db 51%, 
                                        #2cb5e8 75%);
    color:white;
    opacity:0.95;
}

Fuentes Web

Las fuentes web permiten a los diseñadores web para utilizar las fuentes que no están instaladas en el navegador del usuario.

Basta con incluir el archivo de fuente en su servidor web, y se descargará automáticamente al usuario cuando sea necesario.

Las fuentes web se definen utilizando la regla @font-face

@font-face {
    font-family: fuente;
    src: url(archivo_fuente.woff);
    font-weight: bold;
}

Con CSS3, las páginas web pueden finalmente incluir sus propias fuentes.

Transiciones

Las transiciones CSS3 le permite cambiar los valores de propiedad (de un valor a otro), durante un periodo determinado.

Para crear un efecto de transición, debe especificar dos cosas:

  • la propiedad CSS que desea añadir un efecto a
  • la duración del efecto
CSS3
.anim_div:hover {
    opacity: 1;
    background: #1ec7e6;
    width: 130px;
    height: 60px;
    font-size: 55px;
}

.anim_div {
    opacity: 0.4;
    background: #92B901;
    width: 65px;
    height: 30px;
    transition-property: 
        width,height,transform,background,
        font-size,opacity;
    transition-duration: 
        1s,1s,1s,1s,1s,1s;
}

Animaciones

  • CSS3 permiten lograr animaciones con la mayoría de los elementos HTML sin necesidad de utilizar JavaScript o Flash!
  • CSS3 permite a un elemento de cambiar gradualmente de un estilo a otro.
  • Se puede cambiar tantas propiedades CSS que desee, las veces que desee.
  • Para utilizar la animación CSS3, se deben especificar keyframes:
.animated_div {
    animation: animated_div_2 5s 1;
}
@keyframes animated_div {
    0% {  transform: rotate(0deg);
          left: 0px; }
...
    100% { transform: rotate(-360deg);
           left: 0px;  }
}

Media Queries

  • Las 'Media Queries' permiten definir reglas CSS en base a las capacidades del dispositivo que está mostrando la página.
  • Las principales características que utilizan son:
    • alto y ancho de la ventana gráfica
    • alto y ancho del dispositivo
    • orientación del dispositivo
    • resolución
  • Es una técnica muy popular para definir estilos para tablets y celulares

  2. Mary Moe

    ul li a:after {
