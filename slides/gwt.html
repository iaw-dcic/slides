¿Qué es GWT?

GWT es un conjunto de herramientas para crear y optimizar aplicaciones basadas en navegadores.

Su objetivo es permitir el desarrollo productivo de aplicaciones web de alto rendimiento sin que el desarrollador tenga que ser un experto en particularidades de los navegadores, AJAX o JavaScript.

Es de código abierto, totalmente gratuito, y utilizado por miles de desarrolladores de todo el mundo.

¿Qué ofrece GWT?

  • Compilación y Optimización
  • Widgets y Templates
  • Manejo de Eventos no nativos
  • Comunicación Cliente-Servidor
  • Simplifica el desarrollo y la depuración
  • Integración con JavaScript
  • Manejo del historial de navegación (atrás y adelante)
  • Internacionalización

MVP

Para construir aplicaciones de gran escala, el patrón MVP es uno de los que mejor se adapta para su uso con GWT:

  • Desacopla el desarrollo y permitiendo a múltiples desarrolladores trabajar simultáneamente
  • Facilita el testeo de la aplicación, minimizando la necesidad de testear código que se ejecute en el navegador

Super Dev Mode

Permite recompilar rápidamente y ver los cambios reflejados en el navegador

Posibilita depurar el código en Java, pese a que se esté ejecutando código JavaScript, utilizando Source Maps
