Selectores CSS

/* por elemento */
p {
    text-align: center;
    padding: 10px; 
}
/* por id */
#nav {
    text-align: center;
    padding: 10px; 
}
/* por class */
.center {
    text-align: center;
    padding: 10px; 
}

Como incorporar estilos CSS

<head>
  <link rel="stylesheet" type="text/css" href="external.css">

  <style>
    /* internal */
    body {
        background-color: linen;
    }
    h1 {
        color: maroon;
        margin-left: 40px;
    } 
  </style>
</head>

<body>
  <h1 style="color:blue; margin-left:30px;">Inline</h1>
</body>
